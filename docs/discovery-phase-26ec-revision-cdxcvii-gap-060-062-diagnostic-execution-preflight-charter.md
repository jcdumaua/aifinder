# AiFinder Phase 26EC — Diagnostic Execution Preflight Charter

## Purpose

Define the complete fail-closed preflight required before any diagnostic execution may be proposed.

## Required checks

Repository identity, candidate identity, syntax, trace counts, timeout, retry count, no-redirect behavior, reference presence, team-context absence, output allowlist, and fresh human authorization must all pass.

## Boundary

This charter does not authorize execution or a network request.
