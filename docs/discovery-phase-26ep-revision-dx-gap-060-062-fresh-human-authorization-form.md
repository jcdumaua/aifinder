# AiFinder Phase 26EP — Fresh Human Authorization Form

A valid authorization must explicitly approve:

- exactly one candidate invocation;
- at most one read-only request;
- timeout `15` seconds;
- no retry;
- no redirect;
- team context `no`;
- body-free diagnostics only;
- no mutation;
- no operational reactivation.

Silence, prior approval, documentation approval, or patch approval is insufficient.
